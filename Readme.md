# 描述
RSVP范式下的脑电信号，使用EEGNet对信号进行分类

# 环境要求
python >= 3.6  
pytorch >= 1.6.0  
numpy >= 1.18.5  
matplotlib >= 3.1.1  
scikit-learn >= 0.23.1

# 使用方法
## 下载
    git clone https://github.com/Tammie-Li/RSVP-EEGNet.git
## 运行
    python train.py
    python test.py
